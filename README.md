# kshvakov.github.io

Personal GitHub Pages site: [https://kshvakov.github.io](https://kshvakov.github.io)

This repository contains the source for my personal website, including:

- **AI Agent Course** - Hands-on course for building autonomous AI agents in Go
- Other articles and projects (coming soon)

## Structure

- `index.html` - Main landing page with RU/EN language switching
- `404.html` - Custom 404 error page
- `assets/` - CSS and JavaScript files for the site

## Local Development

Simply open `index.html` in a web browser or use a local web server:

```bash
python3 -m http.server 8000
# or
npx serve
```

Then visit `http://localhost:8000`
